## Installation


## Available Commands

### Start Local Server

To run the local server, use the following command:

```bash
yarn start
```

### Create Open Api Types

To generate Typescript types from OpenApi:

```bash
yarn generate-types
```

### Build project

To generate project build:

```bash
yarn build
```
